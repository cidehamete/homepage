Connect to your GitHub repo

One-time setup. The editor commits content.json straight to your repo on save. Cloudflare Pages picks it up and redeploys in ~30 seconds.

  1. Go to github.com/settings/tokens and click Generate new token (fine-grained).
  2. Repository access: Only select repositories → your site repo.
  3. Permissions → Repository → Contents: Read and write. Save.
  4. Copy the token (starts with github_pat_) and paste below.
